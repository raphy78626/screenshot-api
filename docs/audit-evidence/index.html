What you get

Everything an auditor needs,
in a single ZIP

Timestamped PDFs

Every page captured with exact date, time, and version number stamped in the footer. No ambiguity about when the snapshot was taken.

SHA-256 per PDF

Each PDF is hashed after generation. Hashes are recorded in manifest.json so any auditor can run shasum -a 256 and verify independently.

VERIFICATION.txt

Step-by-step verification commands included in every bundle. Auditors can self-verify without any proprietary tools — just a terminal.

Weekly scheduled capture

Set it once — the app exports your selected pages automatically every week and attaches the ZIP to a Confluence page of your choice.

Runs on Atlassian

Forge-native. No external servers, no data egress, no third-party accounts. Everything runs inside your Atlassian tenant.

Structured rendering

Headings, bullets, tables, and code blocks render cleanly in the PDF — not a plain-text dump. Looks professional in an audit package.
